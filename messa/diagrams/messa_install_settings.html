Messa di Voce Computer Setup Instructions
Version 1.0, 3 March 2006 -- golan
(Note that these instructions only discuss how to configure the PC....)

1. Assuming the hosts have given you a brand-new WindowsXP machine.
Make the following changes to the operating system:

  • open My Computer : View : choose Details
  • open My Computer : Tools : Folder Options : General : choose Use Windows Classic Folders
  • open My Computer : Tools : Folder Options : View : Don't automatically search for network folders
  • open My Computer : Tools : Folder Options : View : Display full path in the title bar
  • open My Computer : Tools : Folder Options : View : Show hidden files and folders
  • open My Computer : Tools : Folder Options : View : Don't hide extensions for known file types
  • open My Computer : Tools : Folder Options : View : Don't hide protected OS files
  • open My Computer : Tools : Folder Options : View : Show control panel in My Computer
  • open My Computer : Tools : Folder Options : View : Apply to All Folders : Yes
  • Desktop right click : Display Properties : Themes : choose Windows Classic
  • Desktop right click : Display Properties : Desktop : Color : choose Black
  • Desktop right click : Display Properties : Screensaver : choose None
  • Desktop right click : Display Properties : Screensaver : Monitor Power : set NEVER turn off monitor
  • Task Bar right click : Properties : Task Bar tab : Don't Lock the Taskbar
  • Task Bar right click : Properties : Start Menu tab : choose Classic Start Menu
  • Task Bar right click : Properties : Start Menu tab : Customise..: choose Expand Control Panel
  • Task Bar right click : Properties : Start Menu tab : Customise..: choose Display Admin Tools
  • Task Bar right click : Properties : Start Menu tab : Customise..: choose Show Small Icons
  • Task Bar right click : Properties : Start Menu tab : Customise..: don't Use Personalized Menus
  • Desktop right click : Display Properties : Settings : Advanced : nVidia geForce : Performance & Quality : Image Settings : choose high performance
  • Desktop right click : Display Properties : Settings : Advanced : nVidia geForce : Performance & Quality : OpenGL settings: Vertical Sync: On or Off, whichever works better
  • Desktop right click : Display Properties : Settings : Advanced : nVidia geForce : nView Display Mode : choose Horizontal Span ( requires both LCD and Projector hooked up)

2. Now it's time to install Codewarrior.
There are 3 upgrade packs that need to be installed, and a few important options and actions to pay attention to.

  • Install Winzip : choose Classic
  • extract CW8_DEV.zip
  • CW8_DEV : setup.exe : choose Full Install
  • say Yes to "Setup Environment Variables?"
  • say Yes to "Install Sun JDK 1.3?"
  • say No/Bypass to "Perform Update Check?"
  • don't register, just close (restarts automatically).
  • install Codewarrior 8.1 Update Installer; restart.
  • install Codewarrior 8.2 Update Installer; restart.
  • install Codewarrior 8.3 Update Installer; restart.
  • say YES to "Rebuild Codewarrior Libraries?" Yes! press F7 to compile; wait; F7 again; close Codewarrior.

3. We make sure that both the System and Codewarrior have the correct version of GLU, GLUT.

  • extract glut_swap_in_CW.zip
  • open C:\Program Files\Metrowerks\CodeWarrior\Win32-x86 Support\Libraries\Win32 SDK
  • swap in the new version of glu32.lib
  • open C:\Program Files\Metrowerks\CodeWarrior\Win32-x86 Support\Headers\Win32 SDK\gl
  • swap in the new version of glu.h, glut.h
  • open C:\WINDOWS\SYSTEM32
  • swap in the new version of glu32.dll

4. We install IPP (Intel Performance Primitives).
Note that there is an update pack with an annoying install method:

  • install ipp20win.exe into the location: C:\Program Files\Intel\IPP
  • extract the update pack, w_ipp_pu01_2.0.003.e.zip into a temporary location
  • drag each of the folders in the extracted location, into the corresponding location in C:\Program Files\Intel\IPP.
    This has the effect of replacing the necessary files with their new versions.

5. We install OpenCV:

  • install OpenCV_b3.exe to C:\Program Files\Intel\OpenCV
  • say YES to add OpenCV to the System Path
  • ignore the error, "unable to execute file, RegisterAll.bat"

6. Install the Powermate Knob:

  • install the powermate knob driver
  • install the powermate knob
  • set the knob on the slowest speed
  • in the Powermate control panel, set User Action: Rotate Right to Computer Action: Cursor Right
  • in the Powermate control panel, set User Action: Rotate Left to Computer Action: Cursor Left
  • in the Powermate control panel, set User Action: Click & Rotate Right to Computer Action: Cursor Right
  • in the Powermate control panel, set User Action: Click & Rotate Left to Computer Action: Cursor Left
